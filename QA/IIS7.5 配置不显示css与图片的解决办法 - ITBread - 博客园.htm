IIS7.5 配置不显示css与图片的解决办法

最近在做个网站，我的系统是win7 –64 英文版旗舰版，配置iis成功，可就是不显示图片，css样式无效，感觉很奇怪，网上搜索，没有找到答案，如是重新配置iis，

Internet Information Services→World Wild Web →Common HTTP features ，勾选staticcontent（即中文版的Internet 信息服务→万维网服务→常见HTTP功能-静态内容）重新打开网站即可显示图片并且CSS也可用。
